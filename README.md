# Plagiarism Checker Flask
Plagiarism Checker Flask application is an application built using the Python programming language and using the flask framework technology. This application was built with the aim of assisting in checking the plagiarism of a document or writing. Praise God, this application can be released with the latest version: v1.0.0.
